# The Wealthy Tenet Website

This repository contains the public website for The Wealthy Tenet, a framework for rent-linked asset accumulation.

The website includes:

* an overview of the framework
* the document library and whitepapers
* implementation notes
* articles and essays
* founder background
* contact information

The Wealthy Tenet is currently a research framework and pilot concept. It is not presented as an investment product, financial service, exchange, custodial platform, or policy program.

## Website files

The site is built as a static HTML/CSS website and is intended to be published through GitHub Pages.

Main files:

* `index.html`
* `framework.html`
* `whitepaper.html`
* `implementation.html`
* `articles.html`
* `about.html`
* `contact.html`
* `styles.css`
* `script.js`
* `assets/`
